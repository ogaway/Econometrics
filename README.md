Econometrics
=====
StatsModelsを使ってPythonで計量経済。  

『Rによる計量経済学』のRコードをPythonに翻訳しながら計量経済のお勉強をします。コード以外の部分に関しては基本扱わないので、計量経済学の考え方や数理的裏付け等は本書または別の書籍にあたってください。
